# Titanic Dataset - Basic Exploratory Data Analysis

## Dataset Used
The Titanic dataset from Kaggle was used.  
It contains information about passengers such as age, gender, ticket class, and survival status.

## Analysis Performed
- Loaded the dataset using Pandas
- Inspected dataset structure using head(), shape(), and info()
- Checked and handled missing values
- Calculated basic statistics like mean and standard deviation

## Visualizations
- Survival count of passengers
- Age distribution of passengers
- Survival comparison between male and female passengers

## Insights
- More passengers did not survive than survived
- Female passengers had a higher survival rate
- Most passengers were between 20 and 40 years old

## Machine Learning Usage
This dataset can be used for binary classification models to predict survival based on passenger features.
