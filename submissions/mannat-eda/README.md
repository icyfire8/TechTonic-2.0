# Titanic Dataset Exploratory Data Analysis (EDA)

## Analysis Performed
- Loaded the dataset using pandas
- Checked shape and basic information
- Identified missing values
- Filled missing values in Age using mean
- Filled missing values in Embarked using mode
- Dropped Cabin column due to large number of missing values
- Generated basic statistics
- Created visualizations (Histogram, Bar chart, Scatter plot)

## Visualizations
- Age distribution using histogram
- Gender distribution using bar chart
- Age vs Fare using scatter plot

## Insights
- Most passengers were between 20 to 40 years old.
- Male passengers were more than female passengers.
- Most passengers paid low fares.
- There is no strong relationship between age and fare.
